## What is Social Media Fetcher?

**Social Media Fetcher (SMF)** is a Node.js REST API that automates the bulk backup and synchronization of media assets (images and videos) from major social media platforms.

SMF uses a combination of browser automation (Puppeteer with stealth plugins) and direct REST/GraphQL API consumption to reliably extract media at scale without triggering platform security systems.

## Supported platforms

| Platform | Engine Type | Capabilities |
|---|---|---|
| **DeviantArt** | Hybrid (Puppeteer + REST) | Gallery, header banner, avatar, full-quality downloads |
| **OnlyFans** | Puppeteer (Session Persistent) | Full media feed via smart scroll |
| **Pixiv** | REST (High-Speed) | Illustrations, manga sets, multi-page posts |
| **X (Twitter)** | Puppeteer | User media timeline, dual-method processing |
| **Threads** | Puppeteer + GraphQL | Posts including hidden/spoiler content |
| **TikTok** | Puppeteer | Watermark-free videos, profile photos |
| **Pinterest** | REST | Paginated boards via bookmark cursor |
| **Instagram** | Puppeteer + GraphQL | Stories, Highlights, Posts, Reels |

## How it works

SMF exposes a simple HTTP GET API. You call an endpoint with a `username` (and optional `limit`), and SMF:

1. Launches a Puppeteer browser (or makes direct API calls for REST-engine platforms)
2. Injects your session cookies for authentication
3. Scrolls/paginates through all available media
4. Downloads each file to your configured storage directory
5. Returns a summary response when complete

```bash
# Example: download all DeviantArt media for a user
curl "http://localhost:3000/deviantart/get-all-media?username=someartist&limit=100"
```
