## Prerequisites

- [Node.js](https://nodejs.org/) v18 or later
- Git
- A terminal

<Info>
  `npm install` automatically downloads a local Chrome binary via the `postinstall` script â no separate browser installation required.
</Info>

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/Omarleel/Social-Media-Fetcher.git
    cd Social-Media-Fetcher
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```

    The `postinstall` script runs `puppeteer browsers install chrome` automatically, so Puppeteer has a compatible Chrome binary bundled with the project.

    <Tip>This step can take a minute on first run while Chrome downloads.</Tip>
  </Step>

  <Step title="Create your .env file">
    Copy the example file and fill in the values for the platforms you want to use:

    ```bash
    cp .env.example .env
    ```

    At minimum, set the three core variables:

    ```env
    PORT=3000
    DIR_STORAGE=./storage
    THREADS_DOWNLOAD=5
    ```

    Then add auth credentials for each platform you plan to scrape. See [Configuration](/configuration) for a full reference of every variable.

    <Warning>
      Never commit your `.env` file to version control. It contains session cookies that grant full access to your accounts.
    </Warning>
  </Step>

  <Step title="Start the server">
    ```bash
    npm start
    ```

    You should see:

    ```
    ð Servidor SocialMediaFetcher corriendo en puerto 3000
    ```
  </Step>

  <Step title="Make your first API call">
    Download all downloadable media from a DeviantArt gallery. Replace `USERNAME` with a real DeviantArt username:

    ```bash
    curl "http://localhost:3000/deviantart/get-all-media?username=USERNAME&limit=10"
    ```

    A successful response looks like:

    ```json
    {
      "status": true,
      "profile": {
        "id": 12345678,
        "nickname": "ArtistName",
        "username": "artistname",
        "picture": "https://a.deviantart.net/...",
        "header": "https://a.deviantart.net/...",
        "url": "https://www.deviantart.com/artistname"
      },
      "total_requested": 10,
      "total_processed": 10,
      "downloads": [
        { "status": "downloaded", "fileName": "artwork_title.jpg" },
        { "status": "exists", "fileName": "another_piece.png" }
      ]
    }
    ```

    Downloaded files are saved to `./storage/deviantart/USERNAME/`.
  </Step>
</Steps>

## Available endpoints

All endpoints follow the same pattern: `GET /{platform}/get-all-media`.

| Platform | Endpoint | Key parameters |
|---|---|---|
| DeviantArt | `/deviantart/get-all-media` | `username`, `limit` |
| OnlyFans | `/onlyfans/get-all-media` | `username`, `limit` |
| Pixiv | `/pixiv/get-all-media` | `userId`, `limit`, `mediaType` |
| X (Twitter) | `/x/get-all-media` | `username`, `limit`, `method` |
| Threads | `/threads/get-all-media` | `username`, `limit` |
| TikTok | `/tiktok/get-all-media` | `username`, `limit` |
| Pinterest | `/pinterest/get-all-media` | `username`, `limit` |
| Instagram | `/instagram/get-all-media` | `username`, `limit`, `mediaType` |
